# NodeJS-Weather-App
Weather APP built with NodeJS, Google Map API and Weather API

### Installing

npm install

## Running the App

node app-promise.js -a '355 N Wolfe Rd, Sunnyvale'

## Built With

* Node.js
* axio
* yargs
